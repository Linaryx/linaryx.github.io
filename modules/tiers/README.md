Tier module (фронт):
- Компоненты/композиции для тиров.
- Работа с кэшем `public/api-cache/tiers`.
- Клиентские вызовы Supabase/IVR только с публичными ключами.

Предлагаемое содержимое:
- `composables/` — useTiers, кэш.
- `components/` — таблицы, карточки, контролы.
- `lib/` — помощники форматирования, расчетов.
